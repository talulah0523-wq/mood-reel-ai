# 接住我 · Mood Reel

一个从“此刻的情绪”出发，而不是从电影类型出发的推荐产品原型。

用户只需要描述现在的状态，也可以补充一部曾经打动自己的电影。产品会返回三部电影，并分别解释它们的画面、节奏、人物与故事为什么适合此刻，同时主动提示可能不合适的地方。

## 为什么做

传统电影推荐擅长回答“和这部电影相似的是什么”，却很难回答“现在的我适合看什么”。用户往往要在片单、短评、简介和相似推荐之间反复跳转，才能判断一部电影是否能接住当下的情绪。

## 当前版本

- 自然语言描述心情
- 可选参考电影
- 情绪意图解析
- 三部电影与可解释的推荐理由
- 看前提醒，降低错误期待
- 换一批与反馈闭环
- 响应式中文界面

当前演示使用本地电影样本与可解释匹配逻辑，保证无需密钥即可完整体验。下一阶段可接入大语言模型完成开放式情绪理解，并结合真实影片资料做检索增强。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
pnpm install
pnpm dev
```

打开 `http://localhost:3000`。

## 产品方向

1. 接入模型，将情绪翻译成节奏、情绪浓度、人物距离感等可解释维度。
2. 接入影片资料库，严格约束事实并拓展长尾影片。
3. 保存用户反馈，建立个人化的“被接住”偏好，而不是传统类型画像。

## 技术栈

Next.js、React、TypeScript、vinext、Cloudflare Workers。
